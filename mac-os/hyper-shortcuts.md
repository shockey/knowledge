# Hyper shortcuts

- Hyper + T: Toggle iTerm floating session
- Hyper + {H,J,K,L}: Arrow keys
- Hyper + X: Show or Fill 1Password
- Hyper + D: Things.app quick entry
- Hyper + N: Previous macOS space
- Hyper + M: Next macOS space
